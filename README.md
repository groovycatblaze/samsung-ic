# nfhs-5 health data analysis

an exploratory data analysis and geographic visualization project using the national family health survey (nfhs-5) dataset.

## overview

this project analyzes state-level public health indicators across india to identify regional health disparities and explore the relationship between malnutrition, obesity, hypertension, and high blood sugar prevalence.

## objectives

- clean and preprocess nfhs-5 state-level data
- analyze health indicators using pandas
- visualize trends with matplotlib
- build an interactive folium choropleth map of india

## tools used

- python
- pandas
- numpy
- matplotlib
- folium

- correlation between child anaemia and adult obesity
- hypertension comparison across states
- obesity vs anaemia scatter plot
- interactive choropleth map of obesity prevalence
- state-wise health indicator dashboard
